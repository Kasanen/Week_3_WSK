- Cats

  - PUT /api/v1/cats/:id — only the cat owner or users with role 'admin' can update
  - DELETE /api/v1/cats/:id — only the cat owner or users with role 'admin' can delete

- Users
  - PUT /api/v1/users/:id — only the user themself or 'admin' can update user data
  - DELETE /api/v1/users/:id — only the user themself or 'admin' can delete account
